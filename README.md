## Getting Started



### Prerequisites


* Beautiful Soup
  ```sh
  pip install bs4
  ```
* lxml
  ```sh
  pip install lxml
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/JohnathanGraybeal/SudokuSolver.git
   ```
2. Install pip packages
   ```sh
   pip install
   ```
